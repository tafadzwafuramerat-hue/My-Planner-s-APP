# My Planner's App

<!-- Description -->

My Planner's App is a simple and responsive web application that helps users organize their daily activities. It allows users to create, edit, complete, and delete tasks, write and save notes, view the current date and time, monitor task completion progress, and view a monthly calendar. The application stores data in the browser using Local Storage, ensuring that tasks and notes remain available even after the page is refreshed.

## Features

1. Responsive navigation menu
2. Real-time date and digital clock
3. Add new tasks
4. Edit existing tasks
5. Mark tasks as completed
6. Delete tasks
7. Dashboard displaying:

  * Total tasks
  * Completed tasks
  * Pending tasks
  * Total saved notes
  * Progress bar
8. Monthly calendar with the current day highlighted
9. Create, save, and clear notes
10. Automatic data saving using Local Storage

## Installation

1. Clone or download the project.
2. Open the project folder.
3. Double-click `index.html` or open it in any modern web browser.

No additional software or installation is required.


## Usage

* Open the application.
* Navigate between Home, Dashboard, Tasks, Calendar, and Notes.
* Add and manage daily tasks.
* Mark completed tasks to update the dashboard automatically.
* Save important notes.
* View the current month's calendar.
* All tasks and notes are automatically saved in your browser.
